# Elearning Webapp: an web application which provides courses for users to learn
## Actors: 
 ### Students: register course, watch and learn lessons provides by teachers, interact with teacher by leaving comments, dicuss with other students of the same course
 ### Teachers: create course and manage them, provides lessons and interact with students
## Technologies
### Backend: provides API built with Spring Boot 
### Frontend: ReactJs
### Database: MySQL
